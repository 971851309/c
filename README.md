# C语言

## 1. 项目概述

本仓库为 C 语言课程练习集合。根目录 `变量.cpp` 与 `变量.h` 为同一宏与 `printf` 演示程序；`Project1` 为 Visual Studio 解决方案，内含分支与取值范围等小程序。内容侧重标准输入输出、条件分支与数值类型体验。

## 2. 技术栈

### 前端

无。

### 后端

- C 语言（`stdio.h`、`math.h` 等标准库）

### 数据库

无。

### 开发工具与运维

- Visual Studio（`Project1.sln`、`Project1.vcxproj`）
- 构建产物目录 `Debug`（可执行的 `.exe`、`.pdb` 等）

## 3. 项目架构

各示例**独立**，无共用业务模块：

- `变量.cpp`：单文件 `main`，通过宏定义品牌、日期等字符串与整数，多次 `printf` 输出。
- `Project1/分支结构.cpp`：询问年龄，`scanf` 读入后与 18 比较，输出分支提示。
- `Project1/取值范围.cpp`：使用 `pow` 计算并用 `printf` 输出；同时打印字符与整型表现形式，用于观察类型与范围。

数据流均为：标准输入或常量 → 计算/判断 → 标准输出。

## 5. 核心文件说明

### 项目入口文件和配置文件

- `变量.cpp`：`main` 入口，完成宏常量定义与格式化输出。
- `Project1/分支结构.cpp`、`Project1/取值范围.cpp`：各自包含独立 `main`，可被加入 VS 工程分别编译运行。
- `Project1/Project1/Project1.vcxproj`：工程与源文件组织方式说明（以 VS 打开为准）。

### 核心业务逻辑实现

- `分支结构.cpp`：`scanf("%d", &i)` 后 `if (i >= 18)` 控制输出。
- `取值范围.cpp`：`unsigned int result = pow(2, 32) - 1` 与字符 `c` 的多格式打印。

### 数据模型和 API 接口

- 无网络与数据库；无结构化“数据模型”，仅基础标量与标准库函数。

### 关键组件和服务模块

- 无模块化服务划分；每个 `.cpp` 即一个完整控制台小程序。
